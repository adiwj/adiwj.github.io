**How to Install ?**

Create a new directory from your computer, and then make 2 files for Chrome extension details data :
- manifest.json
- index.html
```
{
  "name": "Dextab",
  "description": "Minimalist New Tab for Chrome",
  "version": "0.1",
  "incognito": "split",
  "chrome_url_overrides": 
  	{
  	   "newtab": "x.html"
  	},
  "permissions": 
  	[
  	   "bookmarks", "history", "storage"
  	],
  "manifest_version": 2
}
```
